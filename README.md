# MoyaMo 모야모
"뭐야, 이 모션? **모야모(MoyaMo)**"는 여행객이나 유학생들이 다양한 국가의 제스처를 쉽고 재미있게 학습할 수 있는 웹 서비스입니다. "손을 묶으면 말을 못 한다"는 이탈리아 속담처럼, 제스처는 많은 문화권에서 의사소통의 중요한 부분입니다. 하지만 같은 제스처도 나라마다 다른 의미를 가질 수 있어 문화적 오해를 일으킬 수 있습니다. 이러한 문제를 해결하기 위해 저희는 'MoyaMo 모야모' 서비스를 개발했습니다.
